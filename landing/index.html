How it works

From webhook to insight in seconds. Here's what happens when a PR is opened.

1

Ingest

PR webhook triggers analysis. Title, description, diff, and metadata are extracted and normalized.

2

Embed

Semantic embeddings generated locally using sentence-transformers. No API cost for similarity search.

3

Compare

Vector similarity search against recent PRs. Potential duplicates surfaced with confidence scores.

4

Analyze

LLM deep dive on code quality, security, and vision alignment. Multi-signal scoring for base detection.

5

Report

Structured output with priority, recommended action, and executive summary. Posted as GitHub comment.
